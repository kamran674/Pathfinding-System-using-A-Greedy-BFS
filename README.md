# Pathfinding-System-using-A-Greedy-BFS
This project implements and visualizes A-star and Greedy Best-First Search algorithms in a dynamic grid environment using Python and Tkinter. It supports obstacle generation, heuristic selection, real-time visualization, and dynamic re-planning. The system is designed for educational and experimental purposes in Artificial Intelligence and pathfinding.

## ✨ Overview

> A modern and interactive **pathfinding visualizer** built using Python and Tkinter.
> It demonstrates how intelligent agents use **A*** and **Greedy Best-First Search** to navigate dynamic environments.

This project is designed for:

* 🎓 AI Students
* 💻 Developers
* 📊 Algorithm Learners
